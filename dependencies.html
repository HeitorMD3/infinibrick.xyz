Dependencies

You will need the following software to compile OpenBlox:

  • Qt 5 qmake
  • Qt 5 Core
  • Qt 5 Gui
  • Qt 5 Network (optional)
  • Qt 5 WebSockets (optional)
  • SDL 2 (optional for server builds)
  • OGRE 1.9 (optional for server builds)
  • OpenAL (optional)
  • ALUT (optional; required if OpenAL is used)
  • OpenSSL
  • zlib
  • libcurl
  • Lua 5.3

The article Building on Windows on the wiki provides instructions for installing the dependencies and building OpenBlox on Windows.
