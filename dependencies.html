Dependencies

OpenBlox has the folowing optional dependencies:

  • ENet (required for networking)
  • curl (required for non-resource assets)
  • Lua (required for scripting)
  • Irrlicht (required for graphics)

These dependencies are not required to compile OpenBlox. To compile OpenBlox Studio, you also need Qt.

The article Building on Windows on the wiki provides instructions for installing the dependencies and building OpenBlox on Windows.
